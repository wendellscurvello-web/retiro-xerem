Como publicar de graça

Para que toda a equipe da gestão acesse o mesmo site e os dados fiquem sincronizados, siga um dos planos abaixo. Ambos são 100% gratuitos.

Plano 1: GitHub Pages + Google Sheets GRÁTIS

Ideal para equipes. O site fica online e os dados são salvos numa planilha do Google que todos podem acessar.

  1. Crie uma conta no GitHub.com (é grátis)
  2. Crie um novo repositório com o nome rama-xerem-retiro
  3. Faça o upload do arquivo index.html para o repositório
  4. Vá em Settings → Pages e ative o GitHub Pages (branch: main, pasta: /root)
  5. Pronto! Seu site estará em https://SEUUSUARIO.github.io/rama-xerem-retiro
  6. Compartilhe esse link com a equipe! 🎉

Dica: Cada pessoa da equipe deve clicar no nome no canto superior direito e se identificar — assim todas as ações ficarão registradas com o nome de cada um!

Sincronizando dados com Google Sheets GRÁTIS

Compartilhe uma planilha do Google entre a equipe. Todos veem os mesmos dados em tempo real.

  1. Compartilhe a planilha de respostas do Google Forms com a equipe
  2. Cada pessoa acessa o mesmo link do site publicado no GitHub Pages
  3. Vá na aba "Importar" e cole a URL do Google Apps Script
  4. Clique em Importar para sincronizar os dados

Cada pessoa importa os dados ao acessar. Os pagamentos são gerenciados manualmente (cada um marca no seu navegador). Para sincronização completa, veja o Plano 2.

Plano 2: Firebase (para sincronia em tempo real) GRÁTIS

Usa o banco de dados gratuito do Google (Firebase). Todos veem e alteram os mesmos dados em tempo real.

  1. Crie uma conta no Firebase (grátis, plano Spark)
  2. Crie um projeto e ative Firestore Database (modo teste)
  3. Ative Firebase Hosting e faça o deploy do index.html
  4. O Firebase salva os dados no banco e todos os usuários compartilham os mesmos dados

Este plano exige configurar o Firebase (adicionar SDK). Posso ajudar se quiser!

Resumo — O que fazer agora?

  1. 📁 Baixe o arquivo index.html deste site
  2. 🐙 Crie uma conta no GitHub (se não tiver)
  3. 📤 Faça upload do index.html no GitHub
  4. 🌐 Ative o GitHub Pages nas configurações do repositório
  5. 📨 Compartilhe o link com a equipe de gestão
  6. 👤 Cada um se identifica clicando no nome no topo
  7. 📋 Todas as ações ficam registradas no Log de Atividades!

Integração com Google Forms / Sheets

Importe automaticamente os inscritos do seu formulário Google. As inscrições de Junho vão para a 1ª Cota, Julho para a 2ª Cota, Outubro para a 3ª Cota.

Código Google Apps Script

Copie e cole no editor do Google Apps Script vinculado à sua planilha.

Passo a Passo

  1. Crie um Google Forms com: Nome, Idade, Data de Inscrição
  2. Na planilha de respostas, vá em Extensões → Apps Script
  3. Cole o código ao lado e salve (Ctrl+S)
  4. Configure o gatilho: "Ao enviar formulário" → função "importarDados"
  5. Autorize o script na primeira execução
